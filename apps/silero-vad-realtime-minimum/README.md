# silero-vad-realtime-minimum

## Getting started

```
git clone https://github.com/proj-airi/candle-examples.git
cd apps/silero-vad-realtime-minimum
```

## Build

```
cargo fetch --locked
cargo clean
```

### NVIDIA CUDA

```
cargo build --package silero-vad-realtime-minimum
```

### macOS

```
cargo build --package silero-vad-realtime-minimum
```
